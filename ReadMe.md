# 🌍 Travel Diaries - Travel Blog Page

## 📌 Project Overview

Travel Diaries is a responsive travel blog website developed to share beautiful travel stories and personal experiences. The website allows users to explore different destinations through attractive blog cards, detailed travel stories and a visual travel gallery.

This project was created as part of the **Codmetric Web Development Internship Task 4**.

---

## 🎯 Objective

The main objective of this project is to develop a travel blog page where users can:

- Explore travel stories and experiences
- View destination details
- Browse travel photos
- Contact through a simple contact form

---

## ✨ Features

### 🏠 Home Section
- Attractive hero section with travel background
- Introduction to travel stories
- Navigation menu for easy access

### 📝 Travel Stories Section
- Three travel blog stories:
  - 🏔️ A Journey Through Northern Pakistan
  - 🌊 Memories From A Beach Escape
  - 🏛️ Exploring Historical Turkey

- View Story functionality to display:
  - Complete travel experience
  - Destination details
  - Travel tips
  - Favorite moments

### 🖼️ Travel Gallery
- Collection of beautiful travel images
- Image hover effects
- Responsive gallery layout

### 📩 Contact Form
- Simple contact form including:
  - Name
  - Email
  - Message

### 📱 Responsive Design
- Mobile-friendly interface
- Compatible with different screen sizes

## ⚙️ How To Run The Project

1. Download or clone the project repository.

2. Open the project folder.

3. Open the `index.html` file in any web browser.

4. Explore travel stories, gallery, and contact section.
